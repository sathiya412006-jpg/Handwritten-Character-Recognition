# Handwritten Character Recognition

## CodeAlpha Internship Project

### Objective
Build a deep learning model to recognize handwritten digits using image processing and Convolutional Neural Networks (CNN).

### Technologies Used
- Python
- TensorFlow
- NumPy
- Matplotlib

### Dataset
MNIST Handwritten Digits Dataset

### Deep Learning Model
- Convolutional Neural Network (CNN)

### Features
- Image Processing
- Digit Recognition
- CNN Training
- Accuracy Visualization
- Sample Prediction
- Graphical Output

### Project Workflow
1. Load MNIST Dataset
2. Preprocess Images
3. Train CNN Model
4. Evaluate Accuracy
5. Predict Handwritten Digits

### Run Command
python handwritten_recognition.py

### Output
The model predicts handwritten digits with high accuracy.
